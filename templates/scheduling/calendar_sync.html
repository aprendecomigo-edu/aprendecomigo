Event Format

Google Calendar events must follow this format to be correctly imported:

  • Title: Student's name, with " - FALTOU" suffix for absences
  • Location: Teacher's name, optionally with "Prof." prefix
  • Description: Class type code (e.g., "MATH101", "ENGLISH_ADV")
