📐 Usage Guidelines

✅ Do's

  • • Use consistent class names from this guide
  • • Apply gradients to text for brand elements
  • • Maintain proper spacing and typography scale
  • • Use glass effects for modern, floating elements
  • • Follow the established color palette
  • • Ensure accessibility with proper contrast

❌ Don'ts

  • • Don't create custom gradient combinations
  • • Don't mix different font families inconsistently
  • • Don't use colors outside the defined palette
  • • Don't override glass effect properties
  • • Don't ignore responsive design principles
  • • Don't sacrifice readability for visual effects
